# Vision

Навчальний проєкт, що реалізує емулятор файлової системи з деревом каталогів.
Демонструє три ключові ООП патерни: Composite (ієрархія), Command (операції з Undo) та Proxy (контроль доступу).

## Мета

- Практика роботи з абстрактними класами та інтерфейсами
- Демонстрація паттернів Composite, Command та Proxy
- Реалізація системи управління доступом на основі ролей
- Практика написання unit-тестів з xUnit та Moq
- Вивчення LINQ запитів та JSON серіалізації

## Цільова аудиторія

Студенти курсу ООП на C# для закріплення знань про design pattern'и та архітектуру.

## Обмеження

- Проєкт не взаємодіє з реальною файловою системою ОС
- Немає графічного інтерфейсу — тільки консольна демонстрація та тести
- Немає мережевих операцій та розподіленого доступу
