# Vision

File System Emulator — навчальний проєкт для демонстрації ООП принципів та design pattern'ів.

## Мета

Реалізувати емулятор файлової системи, що показує:
- Composite pattern для древовидної структури
- Command pattern з можливістю Undo
- Proxy pattern для контролю доступу

Студенти курсу ООП на C#.
