## SOLID Принципи - Підсумок

| Принцип | Реалізація | Приклад |
|---------|-----------|---------|
| **SRP** | ✓ Кожен клас одна відповідальність | FileItem - тільки файл |
| **OCP** | ✓ Розширення без модифікації | Нові команди без зміни CommandHistory |
| **LSP** | ✓ Поліморфна заміщуємість | FileItem та DirectoryItem через FileSystemItem |
| **ISP** | ✓ Специфічні інтерфейси | ISearchable тільки для DirectoryItem |
| **DIP** | ✓ Залежність від абстракцій | Proxy залежить від ICommand |

## Design Паттерни - Підсумок

| Паттерн | Призначення | Реалізація |
|---------|-----------|-----------|
| **Composite** | Древовидна структура | FileSystemItem, FileItem, DirectoryItem |
| **Command** | Операції з Undo | ICommand, CopyCommand, MoveCommand, DeleteCommand |
| **Proxy** | Контроль доступу | FileSystemProxy, перевірка прав |
